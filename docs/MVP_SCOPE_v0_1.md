# MVP Scope v0.1 (Humble Beginning)

## In
- Mobile/iPad-first web app (desktop works, mainly for printing).
- Sport modes: Soccer, Hockey
- Voice input (hold-to-talk) + text input.
- Deterministic play generation (rule-based parser + defaults).
- Diagram signifiers: players (labeled circles), ball/puck, movement arrows (solid), pass arrows (dashed), optional shot arrow (thicker)
- Show Mode (fullscreen).
- Save locally (last 20 plays).
- Export: Download PNG, Share PNG (mobile share sheet), Print cleanly (print stylesheet)

## Out
- Strategy engine (opponent/team strengths)
- Practice plans / decks
- Role-by-role audio
- Team accounts / collaboration
- Animations
- Video ingestion

## Must be true
- Generate a diagram in < 20 seconds.
- Works well on phone + iPad.
- Show Mode is frictionless.
- Share works via texting an image.
- Print is clean on desktop or iPad.
