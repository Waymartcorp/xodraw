# Hockey Mode (v0.1)

## Canvas
Default: FULL_RINK simplified. Zones optional later.

## Labels
LW, C, RW, LD, RD, G (optional F1/F2/F3)

## Supported concepts (v0.1)
Breakout, forecheck shape, cycle, simple PP setup.

## Example prompts
- Breakout right: RD rims to RW on boards, C swings low, RW chips to C.
- Forecheck 1-2-2: F1 pressures, F2 cuts middle, F3 stays high.
- Cycle: LW to C below goal line, C to RW half wall, RW to point.
