# Signifiers (Common Visual Language)

## Entities
- Player: circle + 2–4 character label (e.g., LW, C, RD, GK)
- Ball/Puck: small filled dot
- Goal/Net: simple goal icon
- Field/Rink: simplified outline

## Actions
- MOVE: solid arrow (curved when describing rotation/switch)
- PASS: dashed arrow
- SHOT: thicker arrow toward goal

## Style constraints
- No gradients, no textures
- High contrast
- Consistent stroke widths
- Prefer labels over icons
- Avoid clutter (“spaghetti arrows”)

## Simplicity rule
If too much is described: render the core sequence and put extra detail into notes.
