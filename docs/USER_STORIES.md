# User Stories

## Core (v0.1)
1. As a coach, I can speak a play idea and get a diagram.
2. As a coach, I can add a short text note for constraints.
3. As a coach, I can show the diagram fullscreen between breaks.
4. As a coach, I can share the diagram as an image to a group text.
5. As a coach, I can print the diagram for practice.
6. As a coach, I can save and quickly reopen the last 20 plays.
7. As a coach, I can flip the play left/right with one tap.
8. As a coach, I can simplify a busy diagram with one tap.

## Later (explicitly not v0.1)
- Role-specific audio
- Practice plan decks
- Opponent tendencies -> suggested counters
