# XODraw

XODraw is a mobile/iPad-first web app that turns **voice + text** into **simple, legible positional play diagrams** for youth sports.

**v0.1 focuses on:**
- Soccer + Hockey
- Simple positioning + movement + pass arrows
- Between breaks / mid-game huddle use
- Share (text image) + print

**Core promise:**
> Speak the play → see the diagram → show/share/print instantly.

**Non-goals for v0.1:**
- Not a full playbook platform
- Not video scouting
- Not advanced strategy recommendations
- Not animation

Design principle: **clarity over features**.
