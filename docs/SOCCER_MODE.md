# Soccer Mode (v0.1)

## Canvas
Default view: HALF_FIELD. Toggle: FULL_FIELD. Minimal markings.

## Labels
GK, LB, CB, RB, DM, CM, AM, LW, RW, ST (also D/M/F)

## Supported concepts (v0.1)
Passing pattern, overlap, switch, press trigger.

## Example prompts
- Build out: CB to DM, DM to RB, RB to RW, RW crosses to ST.
- Overlap left: LW holds wide, LB overlaps, pass into space.
- Press their RB: ST angles run, LW jumps, CM steps.
